Easy Watchdog

This app will enable some flows to monitor something you want to get notified when the value or limit is exceeded.

Then cards:

You will wire your devices to report in data with these Then cards

is-alive. Used for sending data to Watchdog to get notification if provided name has not checked in for a predefined time
is-below. Used for sending data to Watchdog to get notification if provided name has a value that is below a predefined limit
is-higher. Used for sending data to Watchdog to get notification if provided name is higher than a predefined limit

When cards:

Alert occurred. Used to get an alert with a alertmessage, and you wire it with the flows of your desire to get the kind of notification you need


Details and examples is documented on the Homepage
